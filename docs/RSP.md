# Таблица RSP

|      |  ПМ  |  АК  |  ЛК  |  ТК  |  ЛА  |  ТА  |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|  ПМ  |  =   |  +   |  ++  |  ++  |  ++  |  ++  |
|  ЛК  |  +   |  ++  |  =   |  +   |  ++  |  ++  |
|  ТК  |  =   |  +   |  =   |  =   |  ++  |  ++  |
|  ЛА  |  ++  |  ++  |  ++  |  ++  |  ++  |  ++  |
|  АК  |  -   |  =   |  --  |  -   |  ++  |  ++  |
|  ТА  |  =   |  ++  |  +   |  ++  |  ++  |  ++  |

Условные обозначения:
+ ПМ - пехота/мясо 
+ АК - антикаваллерия
+ ЛК - легкая каваллерия
+ ТК - тяжелая каваллерия
+ ЛА - легкая артиллерия
+ ТК - тяжелая артиллерия

| Символ | Значение         |
|  ----  | ---------------- |
|   +    | Бафф при атаке   |
|   -    | Дебафф при атаке |
|   =    | Паритет          |
